# Amazon-Sales-Dashboard
Amazon Sales Dashboard built using Power BI to analyze revenue, profit trends, and regional sales performance.

Amazon Sales Dashboard
📊 Project Overview
This project presents an interactive Amazon Sales Dashboard designed to analyze sales performance, revenue trends, and profit distribution across different regions and product categories.
The dashboard helps identify high-performing segments and areas requiring improvement.
🛠 Tools & Technologies Used:
Power BI
Microsoft Excel
Data Cleaning & Transformation
Basic DAX Calculations

📌 Key Metrics Analyzed:
Total Sales
Total Profit
Order Quantity
Profit Margin
Region-wise Sales
Category & Sub-Category Performance
Monthly Sales Trend

📈 Key Insights:
Identified highest revenue-generating region
Analyzed profit distribution by product category
Found seasonal sales trends
Compared high-profit vs low-profit products

🎯 Objective:
To provide business insights that support data-driven decision-making and improve overall sales performance.
